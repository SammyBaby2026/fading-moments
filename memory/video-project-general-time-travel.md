# 视频项目：将军穿越记 (General's Time Travel)

## 项目信息
- **类型**: 喜剧短视频 / 穿越题材
- **时长**: 15-30秒
- **风格**: 古代 vs 现代 反差喜剧
- **生成工具**: 即梦 (Dreamina)
- **更新频率**: 每日1-3条
- **发布平台**: 网站三语版本 + YouTube Shorts

## 核心设定

### 主角
**李将军** (General Li)
- 唐朝/明朝将军（根据不同集数可变）
- 身穿盔甲，手持长剑
- 性格：严肃、正直、讲规矩
- 口头禅："大胆！"、"成何体统！"

### 穿越场景
- 现代都市街头、咖啡厅、地铁、商场
- 突然穿越，一脸懵逼
- 保持古代仪态和行为习惯

### 喜剧元素结构 (15秒公式)

**0-3秒：建立场景**
- 将军出现在现代场景
- 镜头：特写盔甲+现代背景对比

**3-8秒：冲突/误会**
- 将军用古代思维理解现代事物
- 或用古代方式解决现代问题

**8-12秒：反差高潮**
- 将军的"骚操作"
- 现代人的反应（惊恐/无语/配合演出）

**12-15秒：反转/金句**
- 意想不到的结局
- 或将军的霸气回应

## 反差梗素材库

### 古代观念 vs 现代场景
1. **见到美女**：将军行礼"姑娘有礼" → 女生自拍"这COS好帅"
2. **看到汽车**：将军拔剑"妖兽受死" → 司机淡定按喇叭
3. **进入电梯**：将军以为是牢笼 → 现代人挤进来
4. **扫码支付**：将军以为是暗器 → 拿出手机"将军，时代变了"
5. **看到手机**：将军以为是魔镜 → 视频通话吓到他
6. **喝奶茶**：将军以为是毒药 → 喝完后"再来一杯"
7. **坐地铁**：将军以为是传送阵 → 被挤成肉饼
8. **看到警察**：将军以为是捕快 → 主动投案"本将在此"

### 将军经典台词
- "大胆！见到本将还不下跪！"
- "此乃何物？竟能千里传音？"
- "你们这时代的马，怎么不喂草料？"
- "本将一刀可斩十人，你们这是什么刀法？"（看到切肉机）
- "这妖术...竟能瞬间制冰？"（看到冰箱）

## 即梦提示词技巧

### 人物一致性
```
A ancient Chinese general wearing traditional Tang dynasty armor, 
red and gold color scheme, helmet with pheasant feathers, 
handsome face, serious expression, holding a sword
```

### 现代场景描述
```
modern city street / coffee shop / subway station / shopping mall
bright daylight / neon lights at night
crowded with people in modern clothing
```

### 风格控制
```
photorealistic, cinematic lighting, 4K quality
comedy tone, exaggerated expressions
split screen showing contrast
```

## 输出格式要求

生成的脚本必须包含：
1. 中文文案（对白+旁白）
2. 英文文案（用于YouTube）
3. 德文文案（可选）
4. 即梦分镜表（4个镜头）
5. 每个镜头的即梦提示词
6. 建议的背景音乐
7. 视频标题+标签

## 示例剧集

### 第一集：将军与电梯
**中文标题**: 唐朝将军第一次见电梯，以为是传送阵！
**英文标题**: Tang Dynasty General vs Elevator - Time Travel Comedy

**分镜**:
- 镜头1：将军站在电梯口，看着金属门（特写）
- 镜头2：门打开，将军拔剑"何方妖术！"（中景）
- 镜头3：将军走进电梯，门关闭，他惊慌拍打（近景）
- 镜头4：电梯上升，将军失重表情，旁边上班族淡定玩手机（对比镜头）

**金句结尾**: 
- 将军："此传送阵甚慢，不及本将轻功！"
