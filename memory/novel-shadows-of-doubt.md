# 《暗影疑云》(Shadows of Doubt)

## 小说信息
- **类型**: 心理悬疑 / 都市犯罪
- **风格**: 美式硬汉派侦探 + 北欧黑色惊悚
- **目标长度**: 20-30章
- **更新频率**: 每日一章（北京时间早9点）
- **背景**: 虚构美国城市 Ashford（阿斯福德）， rust belt（铁锈地带）工业城市

## 故事大纲

### 核心设定
Ashford，一座被遗弃的工业城市，曾经的钢铁之都如今只剩空洞的工厂和失落的梦想。
15年前，Harrison Steel Mill（哈里森钢铁厂）发生神秘爆炸，23名夜班工人死亡。
如今，当年的幸存者开始一个个"意外"死亡...

 homicide detective **Jack Morrison**（杰克·莫里森）发现这些案件背后隐藏着一个精心策划的复仇计划，而他自己，也是当年事件的亲历者。

### 主要人物

**Jack Morrison** (男，38岁)
- Ashford PD Homicide Detective（凶杀组探员）
- 离异，女儿Emily 12岁随前妻住在芝加哥
- 酗酒，靠威士忌和安眠药度日
- 15年前那场爆炸，他是唯一的生还者

**Sarah Chen** (女，34岁)
- 《Ashford Gazette》调查记者
- 父亲死于15年前的爆炸案
- 独立、尖锐、不妥协
- 与Jack从对立到被迫合作

**Dr. William Harrison** (男，65岁)
- 退休法医，钢铁厂老板William Harrison Sr.的私生子
- 知道15年前的秘密
- 第3章中"意外"坠楼身亡

**The Machinist（机械师）**
- 复仇计划的执行者
- 身份成谜，可能当年受害者的家属
- 精通机械和化学，不留痕迹

2. 死者手机里都有一条未发送的草稿短信："Forgive me"
3. FBI当年的调查报告被密封，原因不明
4. 钢铁厂爆炸当晚的监控录像全部丢失
