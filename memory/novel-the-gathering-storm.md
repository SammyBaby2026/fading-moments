# 《山雨欲来》(The Gathering Storm)

## 小说信息
- **类型**: 心理悬疑 / 都市犯罪
- **风格**: 美式硬汉派侦探 + 黑色电影（中文写作）
- **目标长度**: 20-30章
- **更新频率**: 每日一章（北京时间早9点）
- **背景**: 虚构美国铁锈地带城市 阿什福德 (Ashford)

## 故事大纲

### 核心设定
阿什福德，一座被遗弃的工业城市，曾经的钢铁之都如今只剩空洞的工厂和失落的梦想。
15年前，哈里森钢铁厂（Harrison Steel Mill）发生神秘爆炸，23名夜班工人死亡。
如今，当年的幸存者开始一个个"意外"死亡...

凶杀组探员 **杰克·莫里森**（Jack Morrison）发现这些案件背后隐藏着一个精心策划的复仇计划，而他自己，也是当年事件的亲历者。

### 主要人物

**杰克·莫里森** (男，38岁)
- 阿什福德警局（Ashford PD）凶杀组探员
- 离异，女儿艾米莉（Emily）12岁随前妻住在芝加哥
- 酗酒，靠威士忌和安眠药度日
- 15年前那场爆炸，他是唯一的生还保安

**陈莎拉** (女，34岁)
- 《阿什福德日报》（Ashford Gazette）调查记者
- 父亲马库斯·陈（Marcus Chen）死于15年前的爆炸案
- 独立、尖锐、不妥协
- 与杰克从对立到被迫合作

**威廉·哈里森医生** (男，65岁)
- 退休法医，钢铁厂老威廉·哈里森的私生子
- 知道15年前的秘密
- 第3章中"意外"坠楼身亡

**机械师 (The Machinist)**
- 复仇计划的执行者
- 身份成谜，可能当年受害者的家属
- 精通机械和化学，不留痕迹

### 剧情时间线
- **现在**: 连环死亡事件发生，阿什福德警局展开调查
- **15年前**: 哈里森钢铁厂爆炸，23名工人死亡
- **3个月前**: 第一个"意外"死亡（被判定为自杀）
- **2周前**: 第二个"意外"（车祸）

### 关键线索
1. 每个死者手中都握着一枚生锈的齿轮（来自哈里森钢铁厂）
2. 死者手机里都有一条未发送的草稿短信："Forgive me"（原谅我）
3. FBI当年的调查报告被密封，原因不明
4. 钢铁厂爆炸当晚的监控录像全部丢失
